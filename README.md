# Simple Universal Asset Viewer - SUAVe
---
Preview various types of files often used for Game Development quickly and easily without needing to bloat your projects.
Handles Audio files, 3D Models, and Sprites/Textures. Support for other kinds of files is planned, however if you have a specific file type you want supported, please open an issue here and we will get to it when we can.
SUAVe is donationware, it is available at [https://dr-phantom.itch.io/suav](https://dr-phantom.itch.io/suav) 
We work on it in our spare time between our games and full time jobs, and we have a very small team so please have patience with us!
